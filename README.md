# BBS
**项目简介**

BBS是一个论坛系统，整个项目的构思是维护几个论坛版块（boards）
每个版块像一个分类一样
在指定的版块里面，用户可以通过创建新主题（Topic）开始讨论，其他用户可以参与讨论回复
